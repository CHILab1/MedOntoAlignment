# ICD11

Data from ICD-11 MMS (January 2025) obtained via API retrieve

- `icd11-mms-complete.tsv` ICD-11 MMS - split with all entities 
- `icd11-mms-diseases.tsv` ICD-11 MMS - split with all entities from chapter 1 to chapter 26
- `icd11-mms-locations.tsv` ICD-11 MMS - split with all entities referring to body parts (subsection of chapter 28)
- `icd11-synonyms.tsv` List of synonyms for each ICD-11 MMS category
