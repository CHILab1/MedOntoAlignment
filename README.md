# OntoMedPix 
OntoMedPix: a semantic network of medical ontologies

This repository is arranged as follows:

### annotation-pipeline
contains the code for the proposed annotation pipeline for the alignment of MedPix 2.0 and ICD-11 ontologies

### icd11
contains the obtained data from ICD-11 MMS via API retrieve

### ontologies
contains the build ontologies, query and inference code

_Refer to README files in each folder for further information_
